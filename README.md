# Tunffli
7688 - turn off the light

基於聯發科Linkit Smart 7688 Wifi 路由晶片，建置家居控制系統。

## Linkit 7688 - Web 部分主體

### Web/Tunffli - 7688

FTP 上傳放置於  /www  資料夾下，以Python 指令啟動 tunffli_main.py

前後端: Bootstrap、Ajax、Python(Flask 

功能實現參照:
